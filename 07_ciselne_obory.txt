7. Otázka - Číselné obory
 - zákony:
 	- asociativní - (a + b) + c = a + (b + c)
	- komutativní - a * b = b * a
	- distributivní - a * (b + c) = a * b + a * c

 - složené číslo - má více dělitelů
 - soudělná čísla - mají alespoň dva společné dělitele
 - důkazy o sudosti
 - Gaussova rovina - rovina na které se zobrazují komplexní čísla - x je reálná část a y je imaginární část
 - goniometrický tvar: |z| * (cos$ + i*sin$)
 - algebraický tvar je a1 + i*a2
